# Artifact Knowledge Workbench

A single-page research-style workbench for cultural artifact knowledge construction and visual analytics.

## Stack
- Frontend: Vue 3 + Vite
- Backend: FastAPI

## Run frontend
```bash
cd frontend
npm install
npm run dev
```

## Run backend
```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```
